Pylint

To implement pylint,follow the steps below.

-Make a requirements.txt and write the following:

pytest-cov pytest pycodestyle pytest-pylint

-Right click on docker compose container and create terminal

-Write pip install --user -r requirements.txt and then you are able to do pylint

Glossary

Class: blueprint for the object

Instantiation: making an object

Object: instantiation of a class

Class: blueprint for the object

Namespace: group of defined symbolic names with information about the objects

Constructor: used to initialize an object

Method: functions defined in the body of a class

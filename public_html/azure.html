Azure is a cloud based platform used to host millions
of products and websites, including this very page!
As stated on the Docker page, Azure can be used with
Docker in order to host the images on the cloud, so
anyone can visit or view whatever website or application
you created. In order to do so, first you login to azure
in your terminal using the command:
docker login azure
Once logged in, you can create an ACI context, which will
allow you to run the images on Azure. You use the command:
docker context create aci aciname
This creates the ACI context "aciname", which you can rename.
Then we run the following command to create a container instance:
docker --context myaci run -dp 80:80 dockerhub-username/dockerhub-repo
The context flag tells the terminal to use that context, and the dockerhub
username and repo name is what Azure uses to build its container instance.
You can switch the context and view your containers with the following:
docker context use aciname
docker ps
You should be able to see an address before the port, in this case
"80:80". Pasting this into your browser will allow you to view
your webpage or app online.

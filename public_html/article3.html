Object Oriented Programming

There are four principles of object oriented programming (OOP). They are encapsulation, inheritance, polymorphism, and abstraction.

Encapsulation: restricting access to methods which stops data from being modified. Ex. calculator class.

Inheritance: class inherits properties from another class Ex. Addition and subtraction class inherit properties from calculation class

Polymorphism: using common interface for different data types

Abstraction: showing what's neccesary Ex. users seeing calculator history

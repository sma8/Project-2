Basic Linux commands:

pwd: Shows the path of the working directory
touch: Creates a file
mkdir: Creates a directory
cd: Move into the specified directory
ls: View the contents of the current directory
ls -a: View the contents and hidden files in directory
rm: Deletes files
rm -r: Recursively deletes a folder and its files
mv: Moves and renames files
vi: A basic text editor that is included with Linux

Why should we use Linux Commands?

It's important to know how to use Linux Commands, since many products,
like phones, computers, and fridges use Linux in some manner. Many of
the things that can be done with UI buttons can also be done using the
commands listed previously. Nothing needs to be downloaded to utilize
these features. Everything is included, and knowing the commands can
help you create programs faster, without having to leave the terminal
you are working on.
